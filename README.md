# Everyday 🌟 个人专属高颜值日程表
一款轻量、可爱、无框架的原生前端日程表应用，主打个性化定制与便捷的日程记录，适配桌面+移动端，数据本地持久化不丢失，还贴心标注2026年全量节假日/调休，做你的专属时间小管家～🥳

## 📖 项目介绍
本项目是纯原生前端开发的个人日程表工具，摒弃复杂框架，轻量流畅无依赖。整体采用可爱清新的UI设计，支持主题切换、日程标签化记录、全局备注等核心功能，日历模块精准展示今日、周末、节假日/调休信息，双击快速查看日程、单击编辑日程，操作简单易懂。所有数据通过`localStorage`本地持久化，用户名、主题偏好、日程内容、备注信息全部自动保存，无需注册登录，安全又省心～

## ✨ 核心功能列表
- 🎨 **个性化主题切换**：内置浅粉/浅蓝/浅绿三款可爱主题，支持hover预览主题效果，选择后自动保存偏好
- 📅 **完整日历核心功能**：自由切换月份，精准标注今日、周末，全覆盖2026年法定节假日/调休/特色节日
- ✏️ **便捷日程编辑**：双击日历单元格弹出日程气泡快速查看，单击进入编辑模式，支持多标签（最多3个）+ 文字内容记录
- 🏷️ **自定义日程标签**：标签支持红/蓝/绿/黄/紫5种颜色选择，自定义2字内标签名，可增删标签
- 💾 **本地数据自动持久化**：用户名、主题、所有日程（标签+内容）、备注信息全部本地保存，刷新/重启页面不丢失
- 📝 **全局备注模块**：独立的备注区域，支持随时编辑、保存、取消、清空备注，记录临时小想法超方便
- 📱 **全端响应式适配**：完美适配桌面端、移动端，界面布局随设备自适应调整，操作体验一致
- ❗ **友好的交互提示**：自定义弹窗提示，操作确认、信息提醒清晰明了，避免误操作
- 🧹 **灵活的内容管理**：支持单日期日程全量删除、备注一键清空，轻松管理日程数据

## 📚 技术栈说明
全程采用**原生前端技术栈**开发，无任何第三方框架/库，轻量无依赖，加载速度快：
- **HTML5**：语义化标签搭建页面结构，合理划分功能模块，表单控件实现交互输入
- **CSS3**：CSS变量实现主题切换，Flex布局完成页面适配，伪类/伪元素优化交互样式，圆角/阴影打造可爱视觉效果，媒体查询实现响应式设计
- **JavaScript(ES6+)**：原生DOM操作实现页面渲染，事件处理（单击/双击/hover等）打造流畅交互，日期对象处理日历逻辑，localStorage实现本地数据持久化，自定义封装弹窗/模态框等通用组件

## 💡 小使用提示
1. 首次使用需输入用户名（最多8字），输入后自动保存，后续无需重复输入
2. 双击日历单元格👉快速查看日程气泡，点击空白处/再次双击可关闭气泡
3. 单击日历单元格👉进入日程编辑模式，可添加标签+输入日程文字，最多添加3个标签
4. 主题颜色按钮支持hover预览，点击即可切换，偏好会自动保存
5. 所有数据均保存在**本地浏览器**中，不会上传至服务器，数据安全有保障
6. 备注区域点击即可编辑，完成后点击「完成」保存，「取消」恢复原内容，「删除」一键清空
